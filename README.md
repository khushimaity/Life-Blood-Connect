# LifeBlood Connect – Blood Donation Management System

LifeBlood Connect is a web-based blood donor management platform designed to connect blood donors with emergency blood requests. The system helps hospitals, administrators, and donors coordinate efficiently during urgent blood requirements.

---

## Features

- Donor registration and profile management  
- Emergency blood request system  
- Admin dashboard for managing blood requests and donors  
- Donor availability tracking  
- Donation history management  
- Leaderboard system based on donor activity  
- Secure authentication for different user roles  

---

## User Roles

### Donor
- Register and create a donor profile  
- Update availability status  
- View and accept blood requests  
- Track donation history  

### Admin
- Create and manage blood requests  
- View registered donors  
- Assign donors to emergency requests  
- Monitor donation records

## How to Run the Project

### 1. Install dependencies

Frontend
```
npm install
```

Backend
```
cd backend
npm install
```

### 2. Start backend server
```
cd backend
npm run dev
```

### 3. Start frontend
```
npm run dev
```
